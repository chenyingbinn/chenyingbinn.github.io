# Yingbin Chen - Personal Portfolio

This is the public portfolio site of Yingbin Chen.

The site presents selected research, field experience, project work, and notes around finance, data, and human judgment.

Public site:
https://chenyingbinn.github.io/

## Public Information Boundary

This public repository and website do not include:

- phone numbers
- identity documents
- transcripts
- full resume files
- private certificates
- sensitive account information

Only public-facing portfolio content and contact email are displayed.

## Visual Identity

The favicon uses an abstract signal mark rather than initials, reflecting the site's theme of finance, data, and human judgment.
